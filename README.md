# Euler Laboratory: Circular Motion & The Basel Problem

This project is an interactive visualization tool developed in **Java Swing**. It explores two of Leonhard Euler's most significant legacies: **Euler's Formula** (geometry and complex numbers) and the **Basel Problem** (number theory).

---

## üß† Mathematical Concepts

### 1. Euler's Formula
The position of the red particle is governed by the relationship:
$$e^{i\theta} = \cos(\theta) + i \sin(\theta)$$

Where:
* **$\cos(\theta)$**: Determines the position on the **Real Axis (X)**.
* **$\sin(\theta)$**: Determines the position on the **Imaginary Axis (Y)**.
* As the angle $\theta$ increases, the particle rotates without moving away from the origin, proving that the complex exponential describes a uniform circular motion.



### 2. The Basel Problem
The side panel visualizes the approximation of the famous infinite series:
$$\sum_{n=1}^{\infty} \frac{1}{n^2} = \frac{1}{1^2} + \frac{1}{2^2} + \frac{1}{3^2} + \dots = \frac{\pi^2}{6}$$



Despite being a sum of fractions of integers, the limit converges exactly to a value dependent on $\pi$. The program shows how the progress bar stabilizes near **1.6449**.

* **`paintComponent(Graphics g)`**: The graphical engine that redraws the interface 33 times per second.
* **`ArrayList<Point> trail`**: Stores previous coordinates to render the red trail of the particle.
* **`actionPerformed(ActionEvent e)`**:
    * Increments the angle $\theta$ for constant rotation.
    * Calculates **50 terms** of the Basel series per cycle to show rapid visual convergence in the progress bar.
* **`Timer`**: Controls the screen refresh rate to maintain a fluid animation.
